Installation

Convert Digispark to Motherducker

1. Download and unzip the installation folder
Download for Windows x64
2. Execute the INSTALL.bat file

You can find this file in the unzipped arduino-cli_0.6.0_Windows_64bit folder

3. Install the drivers when the installation window appears
4. After installation of the drivers continue in the terminal window

Type any key to continue the install

5. Plug in (or plug out and in again) the Digispark USB

When the following message appears "> Please plug in the device ..."

That's it! Your Motherducker device is now ready to use.
